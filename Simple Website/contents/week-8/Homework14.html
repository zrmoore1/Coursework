When the box-sizing property is set to content-box, the browser calculates the width without considering the values of the padding and border-width properties. The width is really the width of the content within the box.

Total width = width + padding + border width
Content width = width

If all properties are set to the same values, a content-box will be larger than
a border-box.

When box-sizing is set to border-box, the width property does include both the padding and border-width values. The width is the total width of the box, from the edges of the border.

Total width = width
Content width = width − border-width − padding
