Grid Layout

The CSS3 topic I'm invesigating is the Grid Layout. The Grid Layout is a tool that allows a developer to easily create a responsive layout that changes in according to screen size.

To create a grid, you simply have to set the display property to grid for the parent container. grid-template-columns and grid-template-rows can be used to specify the number and size of rows and columns. grid-template-areas can be used to layout grid items specified with the grid-area property.

/* Base Settings + Mobile */
#sampleGrid{
  display: grid;
  grid-gap: 1em;
  grid-template-areas:
    "header"
    "sidebar1"
    "content"
    "sidebar2"
    "footer";
}
#sampleHeader{
  grid-area: header;
}
#sampleLeftSidebar{
  grid-area: sidebar1;
}
#sampleRightSidebar{
  grid-area: sidebar2;
  background-color: #ccc;
  color: #444;
}
#sampleContent{
  grid-area: content;
}
#sampleFooter{
  grid-area: footer;
}
#sampleHeader, #sampleFooter{
  background-color: #999;
}
.cell{
  border-radius: 5px;
  background-color: #444;
  color: #fff;
  padding: 10px;
}

/* Small Screens */
@media only screen and (min-width: 600px){
  #sampleGrid{
  grid-template-columns: 20% 1fr;
  grid-template-areas: 
  "header header"
  "sidebar1 content"
  "sidebar2 sidebar2"
  "footer footer";
  }
}

/* Medium to Large Screens */
@media only screen and (min-width: 768px){
  #sampleGrid{
    grid-template-columns: 120px auto 120px;
    grid-template-areas:
      "header header header"
      "sidebar1 content sidebar2"
      "footer footer footer";
  }
}
