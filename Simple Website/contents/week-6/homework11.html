Input Attributes

form
Use: Can be used to identify to which form on the page the input element belongs
Types: Any
Example: <input type="text" form="frmHomework11">
pattern
Use: A regular expression that is used to validate formatted text
Types: text, search, url, tel, email, and password
Example: <input type="tel" pattern="\d{3}[\-]\d{3}[\-]\d{4}" placeholder="###-###-####">
step
Use: Can be used to set an increment or valid values for a numerical input.
Types: number, range, date, datetime-local, month, time, and week
Example: <input type="number" step="3">

Input Types

month
Use: Input for month and year
Example: <input type="month" name="dteMonth" placeholder="month, year">
range
Use: Allows user to input a number within an upper and lower bound
Example: <input type="range" min="0" max="100">
url
Use: An input that expects a url. Expects full url, including https://
Example: <input type="url" placeholder="website.com">

Form Elements

output
Use: Displays the output of a javascript calculation
Example: <output for="100 2">
datalist
Use: Displays a list of possible options for an input element
Example:
<input list="animals">
<datalist id="animals">
<option value="Aardvark">
<option value="Anteater">
<option value="Zebra">
</datalist>
